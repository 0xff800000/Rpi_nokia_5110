# Nokia 5110 LCD SPI drivers and apps for Raspberry Pi 3

## Pin connection

+ 5V (e.g. : pin 2) -> VCC
+ GND (e.g. : pin 25) -> GND
+ BCM 10 -> DIN
+ BCM 11 -> CLK
+ BCM 08 -> CE
+ BCM 25 -> DC
+ BCM 24 -> RST

## vcsprint

This small program prints the content of the tty1 virtual console on the LCD display.
